## âš™ï¸ Installation & Setup

### 1. Prerequisites

* Python 3.9+
* MySQL Server (or any SQL-compatible DB)
* Git

### 2. Backend Setup

1. **Clone the repository:**
```bash
git clone https://github.com/BOT-HARI-01/erp.git
cd college-erp-sih/Backend

```


2. **Create and activate a virtual environment:**
```bash
python -m venv venv
# Windows
venv\Scripts\activate
# Mac/Linux
source venv/bin/activate

```


3. **Install dependencies:**
```bash
pip install -r requirements.txt

```


4. **Configure Environment Variables:**
Create a `.env` file in the `Backend/app/core/config.py` directory:
```env
DATABASE_URL=mysql+pymysql://user:password@localhost/college_erp_db
SECRET_KEY=your_secret_key_here
ALGORITHM=HS256
GEMINI_API_KEY=your_google_gemini_api_key

```


5. **Initialize Database:**
```bash
python create_tables.py

```


6. **Run the Server:**
```bash
uvicorn app.main:app --reload

```


*Server will start at `http://127.0.0.1:8000*`

### 3. ML Model Setup

Ensure the trained model artifacts are present for the AI features to work.

1. Navigate to `Academic_Early_Warning_System/model/`.
2. Copy `academic_risk_model.pkl`, `scaler.pkl`, and `model_features.pkl`.
3. Paste them into `Backend/app/ml_models/`.

### 4. Frontend Setup

The frontend is built with static files. You can serve them using any simple HTTP server or by opening `index.html` directly (though a server is recommended for CORS handling).

* **Using Python HTTP Server:**
```bash
cd FrontEnd
python -m http.server 5500

```
* Open your browser and go to `http://127.0.0.1:5500/Frontend/index.html`.
